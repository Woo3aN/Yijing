# 易经占卜桌面软件 — 需求规格说明

## 项目概述

一款运行在 Windows 系统上的易经算卦桌面软件。用户输入想占卜的问题，软件通过周易方法起卦，显示卦象和易经原文解释。用户可选择配置大模型 API 进行 AI 深度解卦。

**目标用户**：计算机大一学生（也是开发者本人）

## 核心功能

### F1. 占卜输入
- 用户可输入任意文字描述想要占卜的问题
- 允许空输入（有时占卜不需具体问题）

### F2. 起卦方法
提供两种起卦方法，用户自选：

| 方法 | 说明 | 特点 |
|------|------|------|
| 三铜钱法 | 模拟扔 3 枚铜钱，共 6 次，逐次出爻 | 有仪式感，逐爻展示动画 |
| 随机数法 | 直接随机生成 6 个爻值 | 快速，一键完成 |

每次起卦产生：
- **本卦**（当前状态）
- **变卦**（如有变爻，代表发展方向）
- 六爻详细信息

### F3. 卦象展示（离线可用）
不依赖任何 API，展示：
- 卦名（如"乾为天"）、卦序（第1卦）、卦符（䷀）
- **卦辞**：卦的核心判断
- **彖传**：对卦辞的阐释
- **象传（大象）**：卦的整体象征
- **爻辞**：每一爻的解释
- **小象**：每一爻的象传

> 所有文本来源于《周易》通行本原文，为公共领域内容。

### F4. AI 解读（需配置 API）
- 支持 DeepSeek 大模型（默认）
- 支持任意 OpenAI 兼容接口（通义千问、智谱 GLM 等）
- 用户填入 API Key 后可用
- AI 结合用户问题 + 卦象信息进行个性化解读
- 无 API Key 时此功能禁用，界面有提示

### F5. 历史记录
- 每次占卜自动保存
- 支持按日期浏览
- 支持按问题关键词搜索
- 可查看每次占卜的完整详情
- 可删除不需要的记录

### F6. 设置管理
- API 密钥（密码框，本地存储）
- API 地址（可自定义）
- 模型名称（可自定义）
- 测试连接功能
- 设置持久化保存

## 非功能需求

### N1. 可用性
- 中文界面，字体清晰
- 界面简洁，操作直观
- 三铜钱法有动画效果，增强仪式感

### N2. 可靠性
- 离线功能（起卦 + 原文）完全不依赖网络
- API 调用失败有友好提示
- 数据存储安全，不丢失

### N3. 分发
- 打包为单个 .exe 文件
- 用户无需安装 Python 或其他依赖
- 支持 Windows 10 及以上

### N4. 性能
- 起卦过程瞬间完成（动画可控制节奏）
- AI 调用超时设为 60 秒
- 历史记录支持 1000 条以上

## 不包含的功能（v1 范围外）
- 蓍草法（大衍筮法）— 可在后续版本加入
- 纳甲、干支等高级易学内容
- 多语言支持
- 导出 PDF
- 网络同步 / 云存储
